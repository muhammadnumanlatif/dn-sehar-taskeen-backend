# Dr. Sehar Taskeen - Project Monorepo

This repository contains the source code for Dr. Sehar Taskeen's web platform.

## Project Structure

- **`next_app/`**: The main frontend application. Built with **Next.js 14** (App Router).
  - Run `npm install` and `npm run dev` inside this folder to start the website.
  - Build output goes to `next_app/out`.

- **`backend/`**: The Node.js Express server.
  - Handles WhatsApp OTP and Email logic.

- **`docs/`**: Project documentation, Guides, and SEO strategies.

- **`_ARCHIVED_VITE_PROJECT/`**: Read-only archive of the initial React + Vite prototype.

## Quick Start (Frontend)

```bash
cd next_app
npm install
npm run dev
```

## Quick Start (Backend)

```bash
cd backend
npm install
npm start
```
